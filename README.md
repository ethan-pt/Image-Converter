## General Information

  Converts PNG to JPG, and JPG to PNG

## Technologies Used

* Python 3.10.0
  * Pillow
  * Tkinter

## Features

* Convert PNG to JPG
* Convert JPG to PNG

Press Browse, navigate to image location, and press convert to whichever format you need.
